# Manual 02: Gestión de usuarios y unidades

## Objetivo

Explicar cómo administrar usuarios del portal y cómo registrar o actualizar unidades del condominio.

## 1. Gestión de usuarios del portal

Este apartado está disponible solo para administradores.

## Qué se puede hacer

- Crear cuentas nuevas.
- Definir si la cuenta será `Administrador` o `Usuario`.
- Editar correo, nombre, teléfono y contraseña.
- Eliminar cuentas cuando ya no se usarán.

## Paso a paso para crear un usuario

1. Entrar al módulo `Configuración`.
2. Ubicar la sección `Usuarios del Portal`.
3. Capturar:
   - Nombre
   - Correo electrónico
   - Teléfono
   - Rol
   - Contraseña
4. Dar clic en guardar.
5. Verificar que el usuario aparezca en la lista.

## Paso a paso para editar un usuario

1. Entrar a `Configuración`.
2. Buscar el usuario en el listado.
3. Dar clic en `Editar`.
4. Modificar los datos necesarios.
5. Guardar cambios.

## Paso a paso para eliminar un usuario

1. Entrar a `Configuración`.
2. Buscar el usuario en la lista.
3. Dar clic en `Eliminar`.
4. Confirmar la acción.

## Validaciones recomendadas

- Confirmar que el nuevo usuario pueda iniciar sesión.
- Confirmar que el rol asignado tenga el nivel correcto de permisos.
- Confirmar que las cuentas de prueba ya no sean visibles si no se usarán.

## 2. Gestión de unidades

Este módulo permite administrar los datos de departamentos o unidades del condominio.

## Qué se puede registrar

- Número de departamento o unidad.
- Torre, edificio o sección.
- Tipo de unidad.
- Nombre de la persona asignada.
- Correo vinculado.
- Cuota ordinaria.
- Cuota extraordinaria.
- Renta de bodega.
- Renta de cajones.
- Número de cajones.
- Número de bodegas.
- Número de jaulas de tendido.
- Estatus de pago.
- Porcentaje de indiviso, si aplica.

## Paso a paso para dar de alta una unidad

1. Entrar al módulo `Gestión de Unidades`.
2. Capturar todos los datos conocidos.
3. Revisar el modelo de cobro visible.
4. Guardar la unidad.
5. Confirmar que aparezca en el listado inferior.

## Paso a paso para editar una unidad

1. Buscar la unidad en el listado.
2. Dar clic en `Editar`.
3. Modificar los datos necesarios.
4. Guardar cambios.

## Paso a paso para borrar una unidad

1. Localizar la unidad en el listado.
2. Dar clic en `Eliminar`.
3. Confirmar la acción.

## Cambio rápido de estatus de pago

El administrador puede actualizar el estatus directamente desde el listado:

- `Pagado`
- `Atrasado`
- `Vacante`

## Qué revisar en la prueba

- Que la unidad se vea en el listado después del alta.
- Que el monto total mensual aparezca correctamente.
- Que el recordatorio de pago mensual esté visible.
- Que editar y borrar funcionen sin error.

## Resultado esperado

Las unidades deben quedar registradas y visibles con toda la información necesaria para cobranza y operación.
