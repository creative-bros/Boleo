# Solicitud para convertir los manuales Markdown a Word

Puedes copiar y pegar el siguiente texto en ChatGPT, Word con Copilot o cualquier herramienta de conversión asistida para obtener versiones `.docx` más explicativas y listas para compartir.

## Prompt sugerido

Convierte los archivos Markdown adjuntos en documentos `.docx` explicativos, claros y profesionales en español. 

Quiero que cada documento:

1. Mantenga el contenido original.
2. Mejore la redacción cuando sea necesario sin cambiar el sentido.
3. Use estilo formal y claro para usuarios no técnicos.
4. Incluya portada simple con el nombre del manual.
5. Agregue tabla de contenido si aplica.
6. Resalte pasos importantes, advertencias y recomendaciones.
7. Conserve listas, títulos y subtítulos bien organizados.
8. Termine con una sección breve de “resultado esperado”.

Los manuales corresponden a un portal llamado `Boleo` para administración condominal.

Los archivos a convertir son:

- `01-acceso-y-primer-ingreso.md`
- `02-gestion-de-usuarios-y-unidades.md`
- `03-cobranza-y-reportes.md`
- `04-amenidades-y-mantenimiento.md`
- `05-pruebas-y-retroalimentacion.md`

Además, genera una versión ejecutiva breve en Word que resuma:

- acceso al sistema,
- roles,
- operaciones principales,
- puntos que deben probar,
- y cómo enviar comentarios.

## Recomendación de salida

Pedir que te entregue:

- Un `.docx` por manual.
- Un `.docx` resumen ejecutivo.
- En lo posible, también PDF final para distribución.
