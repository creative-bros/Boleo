# Manual 01: Acceso y primer ingreso

## Objetivo

Guiar al usuario para entrar al portal Boleo por primera vez, identificar su rol y validar que el acceso funciona correctamente.

## Datos que debes compartir

- URL del sistema.
- Correo de acceso.
- Contraseña temporal.
- Rol asignado: `Administrador` o `Usuario`.

## Diferencia entre roles

### Administrador

Puede crear, editar, actualizar y borrar información operativa del portal.

### Usuario

Puede consultar información y descargar PDFs, pero no puede modificar registros administrativos.

## Paso a paso para ingresar

1. Abrir la URL del portal Boleo.
2. Escribir el correo electrónico asignado.
3. Escribir la contraseña.
4. Dar clic en `Entrar`.
5. Confirmar que el sistema muestra el panel principal.

## Qué validar al entrar

- Que el diseño cargue completo.
- Que el nombre del usuario aparezca en la parte superior.
- Que el rol visible corresponda al asignado.
- Que el menú lateral muestre los módulos esperados.

## Recuperación de contraseña

Si el acceso falla:

1. Dar clic en `Recuperar contraseña`.
2. Capturar el correo electrónico registrado.
3. Capturar el número telefónico registrado.
4. Revisar el enlace de recuperación enviado.
5. Definir una nueva contraseña.

## Creación de cuenta

Si se habilitó el flujo de registro:

1. Dar clic en `Crear cuenta`.
2. Capturar nombre, correo, teléfono y contraseña.
3. Enviar el formulario.
4. Verificar que el sistema permita entrar con la nueva cuenta.

## Problemas comunes

### Las credenciales no son válidas

- Confirmar que el correo esté bien escrito.
- Confirmar que la contraseña coincida exactamente.
- Revisar si la cuenta ya fue restablecida.

### La página abre sin diseño

- Hacer recarga forzada con `Ctrl + F5`.
- Confirmar que se abrió la URL oficial del portal.

### No aparece una opción esperada

- Verificar si la cuenta es `Administrador` o `Usuario`.
- Algunas acciones solo están disponibles para administradores.

## Resultado esperado

El usuario debe poder ingresar, identificar su rol y comenzar a navegar por el sistema sin bloqueo.
